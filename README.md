# 🏦 Banking77 Intent Classification – ML Model & Gradio App

This project implements an **Intent Classification System** trained on the **Banking77 dataset**, enabling fast and accurate
understanding of user queries in banking customer-support scenarios.

✔️ The system is deployed on **Hugging Face Spaces** using **Gradio** as the front-end interface.  
✔️ The ML model itself is hosted on **Hugging Face Models Hub**.  
✔️ All training and preprocessing were performed on **Google Colab**, so no training files are included in this repository.

---

## 🚀 Live Demo (Hugging Face Space)

🔗 **https://huggingface.co/NauRaa/banking77-intent-model**

## 🤖 Model Details

- **Model Name:** `NauRaa/banking77-intent-model`  
- **Framework:** PyTorch  
- **Architecture:** Transformer-based sequence classification  
- **Training Notebook:** Developed entirely on **Google Colab**  
- **Dataset:** Banking77 – 77 banking-related intents  
- **Use Case:** Customer support automation, chatbots, query routing systems

---

## 📱 Features

- Predicts the correct **intent** from user text (e.g., *lost card*, *fraud*, *balance inquiries*…)
- Shows predicted label + confidence score
- Lightweight and fast API using **Gradio**
- Clean & simple UI for end-users
- Fully deployable on Hugging Face Spaces
